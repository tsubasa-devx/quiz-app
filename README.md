# クイズアプリ

シンプルなクイズアプリです。  
問題に回答すると、結果が表示されます。

## 🔗 URL

https://quiz-app-3tij.onrender.com

## 🛠 使用技術

- Node.js
- Express
- JavaScript
- HTML / CSS

## ✨ 機能

- クイズ出題
- 回答送信（POST）
- 正解 / 不正解の判定
- スコア表示

## 💡 工夫した点

- フォーム送信（POST）とサーバー処理の流れを理解
- `req.body` を使って回答データを取得
- `res.redirect` でページ遷移を実装
- クエリパラメータでスコアを管理

## ⚠️ 注意

※無料プランのため初回アクセス時に時間がかかる場合があります
